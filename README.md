# Smart CLI Calculator 

A command-line calculator built in Python that performs arithmetic operations with proper validations and error handling.

## Features
- Supports +, -, *, /, //, **, %
- Accepts integers and decimal numbers
- Prevents division by zero
- Handles invalid input using exception handling
- Runs continuously until user exits
- Maintains history of calculations

##  Concepts Used
- while loop
- conditional statements
- try-except
- input validation
- lists for storing history
